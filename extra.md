ADDITIONAL DEFINITIONS & KEY TERMS
Agent: An entity that perceives its environment and acts to achieve goals
Heuristic: An educated guess or estimate that guides search toward solutions
Informed Search: Search that uses domain-specific information (heuristics)
Uninformed Search: Search that doesn't use domain-specific information
Optimal Solution: A solution with the lowest cost among all possible solutions
Admissible Heuristic: A heuristic that never overestimates actual cost
State Space: All possible configurations a system can be in
Goal State: The desired final configuration
Agent Environment: Everything external to the agent that it can perceive and affect
